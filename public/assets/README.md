# Logo Assets

Place your logo image here to automatically replace the default logo across the entire system.

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- SVG
- WebP

## File Naming
Name your logo file as `logo.png`, `logo.jpg`, `logo.svg`, or `logo.webp`.

## Logo Requirements
- Recommended size: 600x600px or larger for crisp display
- Square aspect ratio preferred
- High resolution for crisp display on all devices
- Transparent background if PNG
- The logo will be displayed prominently:
  - Mobile: 96x96px (h-24 w-24)
  - Tablet: 128x128px (h-32 w-32)
  - Desktop: 144x144px (h-36 w-36)

The system will automatically detect and use your logo file.
